## 문제 상세

### A. 리드 풀 고갈
- 1,008건의 리드에 반복적으로 콜드 메일 발송 중
- 같은 메시지, 같은 타이밍, 같은 풀
- 리드 피로도 관리 없음 (최종 접촉일 필터 없음, 90일 휴면 처리 없음)
- 소프트 바운스 추적 없음
- **리스크:** 도메인 평판 하락, 스팸 신고 가능성

### B. 인바운드 퍼널 단절
- 블로그에 트래픽은 있지만 **전환 메커니즘이 0개**
- 리드 마그넷 없음 (연락처 대가로 다운로드할 자료)
- UTM 파라미터 미설정 (어떤 링크에서 왔는지 추적 불가)
- GA4 전환 이벤트 미설정
- 문의 → 데모 전환 프로세스 자체가 부재

### C. 콘텐츠 생산 중단
- 1월 5건 발행 후, 2~4월 0건
- 연간 콘텐츠 캘린더 (120건 계획)는 존재하나 실행 안 됨
- AI 콘텐츠 파이프라인은 구축 → **소재 발굴/기획 보조 도구로 전환**하여 활용
- 자동 발행(SNS/Webflow)은 미완성이므로 제거, 수동 발행 체제로 전환

### D. 측정 체계 부재
- GSC/GA4 데이터는 CRM으로 들어오지만...
- 확인 불가: "어떤 콘텐츠가 어떤 리드를 만들었는가?"
- 추적 불가: "어떤 메일 시퀀스가 전환했는가?"
- 답변 불가: "리드당 비용이 얼마인가?"

## 축 3: 측정 인프라

### 현재 답할 수 없는 질문들

| 질문 | 답변 가능? | 해결책 |
|------|-----------|--------|
| 어떤 블로그 기사가 리드를 만들었는가? | 불가 | UTM + 리드 소스 추적 |
| 어떤 메일 시퀀스의 전환율이 높은가? | 불가 | GA4 전환 이벤트 |
| 채널별 리드당 비용은 얼마인가? | 불가 | 어트리뷰션 모델 |
| 어떤 리드 마그넷의 ROI가 높은가? | 불가 | 다운로드 → 데모 추적 |
| 콘텐츠 투자가 회수되고 있는가? | 불가 | 콘텐츠 → 리드 퍼널 |

### 구현 계획

```
Step 1: UTM 파라미터 (1주차)
    → 모든 블로그 CTA: utm_source=blog&utm_medium=lead_magnet&utm_campaign={기사슬러그}
    → 모든 메일 링크: utm_source=email&utm_medium=cold&utm_campaign={시퀀스명}

Step 2: GA4 전환 이벤트 (1주차)
    → 이벤트: lead_magnet_download (파라미터: 마그넷명, 기사 소스)
    → 이벤트: demo_request_submit
    → 이벤트: inquiry_form_submit
    → 이벤트: cta_click (파라미터: CTA 유형, 페이지 URL)

Step 3: CRM 어트리뷰션 (2-3주차)
    → Company 모델: firstTouchSource, firstTouchCampaign 필드 추가
    → EmailLog: 전환 이벤트 연결
    → 퍼널 구축: 표시 → 클릭 → 다운로드 → 문의 → 데모 → 계약

Step 4: 대시보드 (3-4주차)
    → CRM Reports 모듈에 퍼널 시각화
    → 주간 자동 Slack 리포트: 소스별 리드 수, 전환율
```

## 요약

### 기억해야 할 3가지

**1. 월 63만 표시 수가 낭비되고 있다.**
> CTR 0.86% → 2.0% = 클릭 2배. 리드 마그넷 추가 = 리드 10배.

**2. CRM은 만들어져 있다. 필요 없는 건 빼고, 핵심만 가동시킨다.**
> 15개 크론잡 가동 중. 소재 발굴 도구 활용 가능. 스크래퍼 준비 완료.
> 미완성 기능(SNS/Webflow 자동 발행)은 제거하고, 리드 전환에 집중.

**3. 콜드 메일 반복 발송을 멈추고, 인바운드 리드를 벌어야 한다.**
> 이번 달 762통 발송 → 전환 0건.
> 유용한 PDF 다운로드 페이지 1개가 콜드 메일 1,000통보다 효과적이다.

### 공식

```
기존 트래픽 (63만 표시 수)
    + CTR 최적화 (0.86% → 2%)
    + 리드 마그넷 (다운로드 도구 5종)
    + 전환 추적 (UTM + GA4)
    + 너처링 자동화 (5단계 메일)
    = 3개월 내 리드 볼륨 10배
```
